不知道何時出生怎麼辦？出生時辰推算功能已推出

人生說明書 · 基本介紹　＃請大家幫忙測試

李伯彥：由於許多朋友不知道自己的出生時間，我根據八字五行和紫微斗數的框架做出權重設計，在人生說明書中推出時辰推算機制。流年分析需要精準時辰，若同學已知自己的時辰，歡迎幫忙測試！

進行方式

  1. 點選：點我推算時辰
  2. 透過 Google 登入（為防止濫用）
  3. 根據題目作答（部分題型可複選）
  4. 看推算結果
  5. 若不正確，請告知實際時辰
  6. 完成後會將推算結果直接帶入輸入資料

大家有空歡迎幫忙測試，看目前的機制到底準不準。

儲存機制

為了答謝大家的幫忙，我設計了儲存機制。登入後可選擇儲存自己和家人的資料，最多五筆。對我來說是很好用，因為我要持續測試、優化，輸入的都快背起來了。

後台分析與改進

此外，我會定期公布後台分析數據，還有我的改進方法給大家參考。

安全性

為了確保安全性，我已設定安控機制，防止大量刷 API 填入假資料。

若有任何不足歡迎大家隨時告知，一起將人生說明書設計得更好，讓更多人都能使用。
