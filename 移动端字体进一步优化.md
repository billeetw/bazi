# 📱 移动端字体进一步优化

## ✅ 优化内容

### 宫位详情面板（移动端）字体调整

#### 1. 基础字体大小

**移动端（< 768px）** → **桌面端（≥ 768px）**：
- `mobilePalaceBody` 基础字体：`text-sm` (14px) → `text-xs` (12px)
- 标题：`text-lg` (18px) → `text-base` (16px)
- 副标题：`text-sm` (14px) → `text-xs` (12px)

#### 2. 战略金句

**移动端** → **桌面端**：
- 标签：`text-xs` (14px) → `text-[10px]` (10px)
- 内容：`text-base` (16px) → `text-sm` (14px)
- 内边距：`p-4` (保持)

#### 3. 宫位解释

**移动端** → **桌面端**：
- 标签：`text-xs` (14px) → `text-[10px]` (10px)
- 内容：`text-base` (16px) → `text-sm` (14px)
- 内边距：`p-4` (保持)

#### 4. 星曜说明卡片

**移动端** → **桌面端**：
- 星曜名称：`text-lg` (18px) → `text-base` (16px)
- 五行标签：`text-xs` (14px) → `text-[10px]` (10px)
- 说明文字：`text-sm` (14px) → `text-xs` (12px)
- 内边距：`p-4` (保持)

#### 5. 空宫提示

**移动端** → **桌面端**：
- 标题：`text-base` (16px) → `text-sm` (14px)
- 说明：`text-sm` (14px) → `text-xs` (12px)

## 📊 字体大小对比表

| 元素 | 移动端 | 桌面端 | 增加幅度 |
|------|--------|--------|----------|
| 基础字体 | 14px | 12px | +17% |
| 标题 | 18px | 16px | +13% |
| 副标题 | 14px | 12px | +17% |
| 战略金句内容 | 16px | 14px | +14% |
| 宫位解释内容 | 16px | 14px | +14% |
| 星曜名称 | 18px | 16px | +13% |
| 星曜说明 | 14px | 12px | +17% |
| 标签文字 | 14px | 10px | +40% |

## 🎯 优化效果

1. **可读性大幅提升**：移动端字体增大 13-40%
2. **层次更清晰**：不同元素的大小差异更明显
3. **触摸友好**：更大的字体减少误触
4. **视觉舒适**：在 6 寸屏幕上阅读更轻松

## 🔍 技术实现

使用 Tailwind CSS 响应式类：
- `text-sm md:text-xs` - 移动端 14px，桌面端 12px
- `text-base md:text-sm` - 移动端 16px，桌面端 14px
- `text-lg md:text-base` - 移动端 18px，桌面端 16px
- `text-xs md:text-[10px]` - 移动端 14px，桌面端 10px
