# 📱 移动端字体优化说明

## ✅ 已优化的内容

### Strategic Panel（十神戰略）组件移动端字体调整

#### Section A: 原廠設定與當前武裝 (The DNA)

**移动端（< 768px）**：
- 标题：`text-base` (16px) → 桌面端：`text-sm` (14px)
- 标签：`text-sm` (14px) → 桌面端：`text-xs` (12px)
- 星曜名称：`text-xl` (20px) → 桌面端：`text-lg` (18px)
- 内容文字：`text-base` (16px) → 桌面端：`text-sm` (14px)
- 内边距：`p-4` (保持，移动端和桌面端一致)

#### Section B: 2026 能量天氣預報 (The Environment)

**移动端（< 768px）**：
- 标题：`text-base` (16px) → 桌面端：`text-sm` (14px)
- 元素标签：`text-sm` (14px) → 桌面端：`text-xs` (12px)
- 进度条高度：`h-3` (12px) → 桌面端：`h-2` (8px)
- 间距：`space-y-2` → 桌面端：`space-y-1`
- 预警文字：`text-sm` (14px) → 桌面端：`text-xs` (12px)

#### Section C: 十神戰略 (The Strategy)

**移动端（< 768px）**：
- 标题：`text-base` (16px) → 桌面端：`text-sm` (14px)
- 年度主旋律标签：`text-sm` (14px) → 桌面端：`text-xs` (12px)
- 年度主旋律内容：`text-base` (16px) → 桌面端：`text-sm` (14px)
- 行动/禁忌清单标签：`text-sm` (14px) → 桌面端：`text-xs` (12px)
- 行动/禁忌清单内容：`text-sm` (14px) → 桌面端：`text-xs` (12px)
- 清单间距：`space-y-2` → 桌面端：`space-y-1`
- 采集标签：`text-sm` (14px) → 桌面端：`text-xs` (12px)
- 文本输入框：`text-base` (16px) → 桌面端：`text-sm` (14px)
- 输入框行数：`rows="4"` (移动端更多行，便于输入)

## 📊 字体大小对比

| 元素 | 移动端 | 桌面端 | 增加 |
|------|--------|--------|------|
| 标题 | 16px | 14px | +14% |
| 标签 | 14px | 12px | +17% |
| 星曜名称 | 20px | 18px | +11% |
| 正文 | 16px | 14px | +14% |
| 清单项 | 14px | 12px | +17% |
| 输入框 | 16px | 14px | +14% |

## 🎯 优化效果

1. **可读性提升**：移动端字体增大 11-17%，更容易阅读
2. **触摸友好**：更大的字体和间距，减少误触
3. **视觉层次**：保持桌面端的视觉层次，但整体放大
4. **响应式设计**：使用 Tailwind 的 `md:` 断点，自动适配不同屏幕

## 📝 技术实现

使用 Tailwind CSS 响应式类：
- `text-base md:text-sm` - 移动端 16px，桌面端 14px
- `text-sm md:text-xs` - 移动端 14px，桌面端 12px
- `text-xl md:text-lg` - 移动端 20px，桌面端 18px
- `space-y-2 md:space-y-1` - 移动端更大间距
- `h-3 md:h-2` - 移动端更粗的进度条
