# 新增文章指南（無需 CMS）

文章採**靜態 HTML**，不需後台 CMS，直接新增檔案即可。

---

## 步驟

### 1. 建立文章頁面

在 `articles/` 目錄新增 HTML 檔，例如 `articles/新文章-slug.html`。

**複製模板**：複製 `articles/2026-liunian-guide.html`，修改以下區塊：

```html
<!-- head 內 -->
<meta name="description" content="你的文章摘要，約 150 字" />
<title>文章標題｜人生說明書</title>
<link rel="canonical" href="https://www.17gonplay.com/articles/新文章-slug.html" />
<meta property="og:title" content="文章標題" />
<meta property="og:description" content="摘要" />

<!-- JSON-LD 內 -->
"headline": "文章標題",
"description": "摘要",
"url": "https://www.17gonplay.com/articles/新文章-slug.html",

<!-- body 內 -->
<h1>文章標題</h1>
<!-- 正文用 h2、h3、p 標籤 -->
```

### 2. 更新文章列表

在 `articles.html` 的 `<div class="space-y-4">` 內新增一筆：

```html
<a href="articles/新文章-slug.html" class="block glass p-5 rounded-xl ...">
  <span class="text-xs ...">基本介紹 或 深度分析</span>
  <h2 class="...">文章標題</h2>
  <p class="...">摘要 1–2 行</p>
</a>
```

### 3. 更新首頁精選（可選）

若要在首頁「延伸閱讀」顯示，在 `index.html` 的精選區塊加一欄，或替換現有項目。

### 4. 更新 sitemap.xml

在 `</urlset>` 前新增：

```xml
<url>
  <loc>https://www.17gonplay.com/articles/新文章-slug.html</loc>
  <lastmod>2026-02-16</lastmod>
  <changefreq>monthly</changefreq>
  <priority>0.7</priority>
</url>
```

---

## 命名建議

- 檔名：英文小寫、連字號，如 `2026-liunian-guide.html`
- 避免中文檔名（部分環境支援不佳）

---

## 未來若需 CMS

可考慮：
- **Cloudflare D1**：文章存 DB，Worker 動態渲染
- **Markdown + 建置**：文章寫 `.md`，build 時轉 HTML
- **Headless CMS**：如 Strapi、Contentful 等，API 取內容後渲染

現階段靜態 HTML 已足夠，部署簡單、效能佳、利於 SEO。
