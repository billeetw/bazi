# 星曜能量節奏（動星／智星／穩星…）分析規則

## 問題說明

若以**全盤 12 宮**的星曜做六類統計，每張紫微命盤的星曜集合相同（14 主星 + 8 輔星），統計結果會人人一樣，例如固定出現：

- 動星：5　智星：5　穩星：5　權星：5　財星：4　情緒星：2

## 修正方式（已實作）

**星曜能量節奏**改為只依「**命宮三方四正**」四宮的星曜計算：

- **命宮**（ming）
- **財帛宮**（cai）
- **官祿宮**（guanglu）
- **遷移宮**（qianyi）

這四宮的星曜組合會因命盤而異，因此動星／智星／穩星／權星／財星／情緒星的**數量與比例會隨人不同**。

- 若這四宮完全沒有星（異常情況），則 fallback 為全盤星曜，避免出現「無星曜資料」。

---

## 統計規則（供你檢查）

### 1. 資料來源

- `starByPalace`：來自 `buildAssembleInput(chartJson)`，由 `chartJson.ziwei`（starByPalace / mainStars / palaces）解析。
- 用於節奏的星名：僅取自上述四宮（命、財帛、官祿、遷移）的 **主星 + 輔星**（不計煞星）。

### 2. 星曜 → 群性標籤（STAR_PERSONALITY_MAP）

| 群性   | 主星 | 輔星 |
|--------|------|------|
| 動星   | 七殺、破軍、貪狼、廉貞 | 天馬 |
| 智星   | 天機、巨門、天梁 | 文昌、文曲 |
| 穩星   | 武曲、天府、天相 | 左輔、右弼 |
| 權星   | 紫微、廉貞、太陽 | 天魁、天鉞 |
| 財星   | 武曲、天府、太陰 | 祿存 |
| 情緒星 | 太陰、天同 | （無） |

一顆星可屬多類（例如武曲＝穩星+財星、廉貞＝權星+動星），統計時**每個標籤各加 1**。

### 3. 計數方式

- `calculateStarGroupStats(starNames: string[])`：依 `worker/src/lifebook/starPersonalityMap.ts` 的 `STAR_PERSONALITY_MAP` 把每個星名對應到標籤，再累加六類次數。
- 星名須為**中文**（如「天機」「文昌」），且存在於 `STAR_PERSONALITY_MAP` 才會被計入；未在 map 內的星不影響六類數字。

### 4. 顯示邏輯

- 文案與數字來自 `buildStarEnergyRhythmBlock(starNames, patternPalace)`（`worker/src/lifebook/starGroupNarrative.ts`）。
- 顯示「動星：n　智星：n　…」及「其中 X 比例最高」、Top 2 群的 message/action，皆基於上述**命宮三方四正**的 `starNames`（或 fallback 之全盤星曜）。

## 驗收建議

1. 用兩張以上不同命盤（例如不同農曆生日）產出 s03，確認「星曜能量節奏」的六類數字**不再人人相同**。
2. 手動檢查：從命盤查出命宮、財帛、官祿、遷移四宮的主星與輔星，對照上表手算六類次數，應與輸出一致。
