# Cloudflare Pages 部署失敗排查

當出現 `Failed to publish your Function. Got error: Unknown internal error occurred`：

## 1. 直接重試
有時是暫時性錯誤，再執行一次：
```bash
npm run deploy
```

## 2. 加上 --commit-dirty=true（忽略未提交變更）
```bash
npx wrangler pages deploy . --commit-dirty=true
```

## 3. 只部署 Pages（略過 Worker）
```bash
npm run deploy:pages
# 或
npx wrangler pages deploy .
```

## 4. 檢查詳細 log
```bash
# 查看最新部署 log
tail -100 /Users/bill/.wrangler/logs/wrangler-2026-02-16_03-14-25_206.log
```

## 5. 檢查 Cloudflare 狀態
https://www.cloudflarestatus.com/

## 6. 若持續失敗
- 至 Cloudflare Dashboard → Workers & Pages → 你的專案 → Deployments
- 檢查最近是否有成功部署，或全部失敗
- 若有 Git 整合，可試著用 GitHub push 觸發部署
- 聯繫 Cloudflare Support
