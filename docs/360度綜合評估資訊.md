# 360 度綜合評估資訊

> 為評估方提供的技術棧、資料結構與第三方服務現狀整理。

---

## 1. 技術棧詳情 (Tech Stack)

### 前端
- **主體**：純 HTML + Vanilla JS（無 React/Vue 框架）
- **部分使用 React**：`StartupSequence.tsx`（啟動動畫）、少數入口；Vite 以 `@vitejs/plugin-react` 編譯
- **樣式**：Tailwind CSS（`css/tailwind-input.css` → `dist/tailwind.css`）、`css/main.css`
- **打包**：Vite 7.x，多個 config：
  - `vite.main.config.ts` → `dist/app.js`（主站）
  - `vite.divination.config.ts` → `dist/divination.js`（占卦頁）
  - `vite.expert-admin.config.ts` → `dist/expert-admin.js`（專家後台）

### 後端與資料庫
- **靜態 + API**：Cloudflare Pages（靜態檔 + Pages Functions）
- **API Worker**：`worker/` → Cloudflare Worker (`bazi-api.billeetw.workers.dev`)
- **資料庫**：Cloudflare D1（`consult-db`，SQLite 相容）
- **快取**：Cloudflare KV（`CACHE` binding），供 content/2026、estimate-hour-stats 等

### 部署
- **平台**：Cloudflare Pages（專案 `bazi`）
- **網域**：https://www.17gonplay.com
- **流程**：`./deploy.sh` → build bundles → D1 migrations → Worker deploy → Pages deploy

## 2. 資料結構 (Data Schema)

### 用戶資料與出生時間

**users**
```sql
id TEXT PRIMARY KEY,
provider TEXT,           -- 'google' | 'facebook'
provider_user_id TEXT,
email TEXT,
name TEXT,
created_at TEXT
```

**user_charts**（每人最多 5 筆）
```sql
id, user_id, label, birth_date, birth_time, gender, sort_order, created_at
```

**出生時間**：`birth_date`（YYYY-MM-DD）、`birth_time`（時辰或 HH:mm）

### 占卦紀錄

**divination_logs**
```sql
id, created_at, user_id (可選),
question, mood,
primary_index, transformed_index, mutual_index,
lines_json TEXT,       -- [7,8,9,7,8,6] 六爻數值
changing_lines_json,  -- [2,5] 動爻索引
source, feedback_rating, feedback_text, feedback_at
```

**lines_json 範例**：`"[7,8,9,7,8,6]"`（6/7/8/9 表老陰少陽少陰老陽）

### 諮詢報名

**consultations**
```sql
id, created_at, updated_at,
name, email, phone, tax_id,
birth_info, topics, topic_extra, bank_last5,
payment_method, payment_status, source, activity_id
```

### 內容庫：384 爻解釋

- **儲存**：`data/iching/lines-384.json`（靜態 JSON）
- **結構**：`{ "lines": { "1": [...6爻], "2": [...], ..., "64": [...] } }`
- **單爻格式**：
  ```json
  {
    "yang": 1,
    "correct": 10,
    "resonance": -15,
    "base": 0,
    "mag": 1,
    "text": "潛龍勿用。",
    "hint": "潛龍勿用：初期壓力大，宜觀望。"
  }
  ```
- **卦辭/彖象**：`data/iching/hexagram-commentaries.json`
- **來源**：`hexagrams.json` + `scripts/build-lines-384.js` 產生

### 郵件（Resend）
- **狀態**：已串接
- **用途**：諮詢報名成功後寄送確認信
- **設定**：`RESEND_API_KEY`、`CONSULT_FROM_EMAIL`（Cloudflare Pages 環境變數）
- **程式**：`functions/api/consultation.js` → `sendConfirmationEmail()`
