## 八、流年／運／日／時 星曜（iztro 有定義，通常用於流年盤）

這些是同一組星曜在不同時間維度的變體，key 為 yunkui、liukui、yuekui、rikui、shikui 等，對應的繁體名相同（天魁、天鉞、文昌、文曲、紅鸞、天喜、祿存、擎羊、陀羅、天馬）。  
若命盤主畫面不顯示，可暫不補 content；若未來要支援流年盤，再補。

## 十、實作建議

1. **Worker**：在 `extractZiweiMainStars` 中，將 `palace.adjectiveStars` 一併加入 `mainStars`，前端才會收到雜曜。
2. **content-zh-TW.json / content-en.json**：依上表補齊 `stars` 的 key 與解釋。
3. **content-utils.js**：更新 `STAR_FALLBACK_ZH`、`STAR_FALLBACK_EN`，加入上述星曜的 fallback 文案。
4. **star-map.ts**：若使用 en-US，補上雜曜的 `ZH_STAR_TO_EN` 對應。
