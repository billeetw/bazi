# data/wuxing 知識庫來源與藏干權重

## branches.json — 地支藏干

藏干權重採「本氣／中氣／餘氣」比例，對照《淵海子平》《三命通會》等主流子平派：

| 地支 | 藏干 | 權重依據 |
|------|------|----------|
| 子 | 癸×1 | 子水，本氣癸 |
| 丑 | 己0.6, 癸0.3, 辛0.1 | 丑土，本氣己、中氣癸、餘氣辛 |
| 寅 | 甲0.6, 丙0.3, 戊0.1 | 寅木，本氣甲、中氣丙、餘氣戊 |
| 卯 | 乙×1 | 卯木，本氣乙 |
| 辰 | 戊0.6, 乙0.3, 癸0.1 | 辰土 |
| 巳 | 丙0.6, 庚0.3, 戊0.1 | 巳火 |
| 午 | 丁0.7, 己0.3 | 午火（丁己） |
| 未 | 己0.6, 丁0.3, 乙0.1 | 未土 |
| 申 | 庚0.6, 壬0.3, 戊0.1 | 申金 |
| 酉 | 辛×1 | 酉金 |
| 戌 | 戊0.6, 辛0.3, 丁0.1 | 戌土 |
| 亥 | 壬0.7, 甲0.3 | 亥水 |

若採用他派（如《滴天髓》簡化藏干），可另建 `branches-v2.json` 供切換。

## month_branches.json — 月令乘數

月令當令五行 ×1.3、失令約 ×0.9–1.1，為設計參數（非古籍定量）：

- 寅卯木月：木×1.3
- 巳午火月：火×1.3
- 申酉金月：金×1.3
- 亥子水月：水×1.3
- 辰未戌丑土月：土×1.3

見 `docs/五行架構整體評估.md` 中「科學性」說明。
