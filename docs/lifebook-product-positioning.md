# 命書產品定位（對外／對內一句話）

## 核心陳述

> **我們不販售「鐵口直斷的吉凶預測」，而是描述命盤結構與慣性，並提供可選擇的行動路徑。**

讀者拿到的是：**發生了什麼處境 → 可能為何如此 → 若延續會怎樣 → 可以怎麼調整**，而不是單一宿命答案。

## 與功能的對齊

| 模組 | 定位 |
|------|------|
| 十二宮敘事 | 領域慣性與提醒，非斷語。 |
| 模組二（時間／疊宮） | 主線、壓力源、節奏，輔以 **from→to** 等可核對資訊（技術版）。 |
| **S22 結構線** | 對宮「底子與失衡」的生活化描述（情境→因果→未來→行動）。 |
| **S23 轉化流** | 資源／努力如何「接得上結果」的生活化描述，同樣附行動建議。 |
| 技術版（`output_mode: technical`） | **諮詢／內部**用，還原參數與骨架，不取代讀者版敘事倫理。 |

## 合規與溝通

- 對外素材避免「百分之百」「必發生」等表述。  
- 強調 **參考與自我覺察**，重大決策仍應搭配現實資訊與專業協助。
