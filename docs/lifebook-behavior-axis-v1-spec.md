# Behavior Axis v1：語氣偏移 × 星曜衝突 × 循環鏈（規格）

> **目標**：不新增段落、不露框架名詞，只做**微偏移**；讓同一套 H3／H4 **讀起來像這個人**。  
> **程式參考**：`worker/src/lifebook/s17/palaceNarrative/behaviorAxisV1.ts`（`getBehaviorAxis`、偏移句、衝突表、Loop 示範句）。  
> **命宮地支來源**：與 `readZiweiSoulBranch(chartJson)`／`NormalizedChart` 一致，取 **命宮地支** 決定 Axis。

與 **[BM3 總覽](./lifebook-behavior-axis-tone-and-loop.md)** 並讀。

---

## 一、Axis 定義（由命宮地支）

`BehaviorAxis = "dynamic" | "relational" | "introverted"`

| 分類 | 地支（四組各四個） | 意象 |
|------|-------------------|------|
| **dynamic**（驟馬） | 寅、申、巳、亥 | 動、快、外、變 |
| **relational**（桃花） | 子、午、卯、酉 | 人、互動、氣氛 |
| **introverted**（孤獨） | 辰、戌、丑、未 | 收、慢、內、穩 |

預設：不在驟馬／桃花 → **introverted**。

---

## 二、偏移原則（不改核心語義，只改「落點」）

| Axis | 偏移方向 | 核心詞（著色用） |
|------|----------|------------------|
| dynamic | 動、快、外、變 | 動起來、先試、轉場、節奏快 |
| relational | 人、互動、氣氛 | 看人、氣氛、關係、互動 |
| introverted | 收、慢、內、穩 | 先收、想清楚、在自己裡面、慢一點 |

---

## 三、句型偏移模板（v1 固定句）

### 3.1 Headline／斷語

在基準句後接（**不露框架**）：

- **dynamic**：`…，而且不太會停在原地太久`
- **relational**：`…，也會同時在看人與氣氛`
- **introverted**：`…，而且多半會先收在自己裡面處理`

### 3.2 DecisionPatterns（H3）

每宮**最多加一句**，不要三句都加（三句擇一合併或另起一句由產品決定；程式提供**單句**）：

- **dynamic**：`你不太會等到完全穩才動，通常會先試一段再調整`
- **relational**：`你很少只看事情本身，也會同時評估人與關係`
- **introverted**：`你會先在心裡想清楚一輪，再決定要不要動`

### 3.3 Pitfalls（H4）— **首選注入點**

- **dynamic**：`這樣的節奏久了，很容易還沒站穩就已經換場`
- **relational**：`這樣下來，很容易為了關係調整太多自己`
- **introverted**：`久了會變成卡在裡面，想動但啟動不了`

### 3.4 Turn（轉個念）

- **dynamic**：`先慢一點，確認方向，再動會更穩`
- **relational**：`先分清楚這是事情還是關係，再決定怎麼回應`
- **introverted**：`先把想的說出來一小步，再決定要不要動`

---

## 四、注入規則（硬限制）

| 鍵 | 值 |
|----|-----|
| `maxPerPalace` | 1～2 次偏移（建議 2 為上限） |
| `priority` | `pitfall` → `decision` → `headline` |
| `avoidSections` | **H2 `structuralSummary` 不動** |

Tone layer 整合時需 **reject**：含禁用詞、句過長、同一段已偏移兩次仍要加。禁用句式示例：

- 「你是那種會…的人」
- 「你的命格是…」
- 「因為你屬於…」

---

## 五、示範（福德 H3 一句）

**原句**：累或亂的時候，你很少直接處理事情，而是先做一點讓自己好一點的事。

- **dynamic**：…，而且不會停太久，很快會再動起來。  
- **relational**：…，也會看氣氛與關係再決定怎麼動。  
- **introverted**：…，而且多半會先在自己裡面消化。

（實際合併字數需過 `sentenceTooLong` 裁切。）

---

## 六、進階（可選）

若同時命中 **dynamic** 且盤上具 **天馬** 相關訊號，可只調 **語氣強度**（不新增句子）。實作見程式註解位。

# 星曜 × Behavior Axis 衝突表（v1）

**目的**：內在結構（星）與外在行為軸（地支）不一致時，加**一句拉扯感**，不變結論、不加段落。

**原則**：不改結論、不新增段落、只加一句「內外落差」。

## 高價值條目（第一批）

| 星 | Axis | 拉扯句（加在 H4 或斷語後） |
|----|------|---------------------------|
| 太陰 | dynamic | 你其實需要時間感受與消化，但生活的節奏常把你往外推，很難真的停下來。 |
| 武曲 | introverted | 你會把事情扛起來，但不太會把壓力往外放，很多東西最後變成自己消化。 |
| 天機 | dynamic | 你腦中其實會先跑很多版本，但現實常讓你來不及想完就已經做了。 |
| 巨門 | relational | 你很會看人，但也容易在互動裡多想，關係一靠近就開始出現內在拉扯。 |
| 廉貞 | introverted | 你的情緒其實很強，但多半不會直接表達，而是壓在裡面慢慢累積。 |
| 貪狼 | introverted | 你其實有很多想要的東西，但不一定會往外拿，常常變成在心裡反覆拉扯。 |

**匹配**：主星陣列含該星名 + 當前 Axis 一致 → 取**一條**（規則帶 `priority`，數字小先）。

**注入控制**

| 鍵 | 建議值 |
|----|--------|
| `maxPerPalace` | 1 |
| `onlyIfStrongStar` | 主星或高權重星 |
| `priorityPalaces` | 命宮、福德宮、夫妻宮 等 |

**紅線**：不講「因為你是太陰＋驟馬」、不多條疊加、非每宮必打。

# H3 × H4 × 循環鏈 Loop（v1）

**目的**：讓讀者看到 **怎麼做 → 會怎樣 → 為何又繞回來**；Loop **不是**重複 H3／H4，而是 **「一直在發生」** 的一句。

## 句型模板

- **A**：`你會先X → 這會讓Y發生 → 所以你最後又回到X`
- **B**：`你會先X → 你其實知道Y → 但最後還是會回到X`
- **C**：`你想要X → 但會做Y → 所以一直卡在中間`

## 示範句（可當預設庫，每宮最多 1 句）

| 宮位 | Loop（示意） |
|------|----------------|
| 命宮 | 所以你最後還是會回到那個「比較像你」的選擇，只是通常已經繞了一圈。 |
| 財帛宮 | 所以你會在「穩」與「其實可以再往前一點」之間反覆拉扯。 |
| 福德宮 | 所以你會一直在「先讓自己好一點」與「事情其實還在」之間循環。 |
| 夫妻宮 | 所以很多話你其實知道要說，但會拖到關係撐不住才說出口。 |

**控制**：`maxPerPalace: 1`；優先宮：命、福德、財帛、夫妻。

**不要**：每宮都加、解釋「這是循環」、說教口吻。

---

## 與衝突合併（高階）

例：**太陰 × dynamic**：你會先收住自己 → 但現實會推著你動 → 所以你一直在「還沒準備好」與「已經在動」之間來回。

（實作時與 Loop、衝突句 **三者取預算**，避免堆疊。）
