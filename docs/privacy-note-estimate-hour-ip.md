# 隱私說明：推算時辰功能之 IP 記錄（階段三 3.1）

若網站有隱私政策／隱私說明頁面，請加入以下內容（或改寫後納入既有條文）。

---

**推算時辰功能之 IP 記錄**

- 當您使用「推算時辰」功能時，系統會記錄您當次請求的 **IP 位址**（由網路環境提供，例如 `CF-Connecting-IP`）。
- **用途**：僅用於防濫用、異常行為分析與服務維運（例如辨識同 IP 多帳號異常使用），不會用於行銷或對外提供。
- **保留期限**：與推算時辰紀錄一併保存，依本站資料保留政策刪除或匿名化。
- **不提供 IP**：若您所處網路未提供 IP，該筆紀錄的 IP 欄位將為空，不影響推算結果。

---

以上說明可依實際隱私政策格式調整；若有法規（如 GDPR）要求，請另行評估同意機制與保留期限。
