# 🧪 功能测试快速开始

## 一键启动测试

```bash
# 启动测试服务器
./start-test-server.sh
```

服务器启动后会自动打开浏览器，访问 http://localhost:8000/index.html

## 快速检查模块加载

1. 打开浏览器开发者工具（F12）
2. 切换到 Console 标签
3. 复制 `browser-console-check.js` 的内容并粘贴执行
4. 查看检查结果
