# 🏗️ 架构优化指南

## 📋 快速使用

### 1. 检查文件引用

```bash
bash scripts/check-references.sh
```

检查 `index.html` 中引用的所有 JavaScript 文件是否存在。

### 2. 更新版本号

```bash
# 使用时间戳
bash scripts/bump-version.sh $(date +%s)

# 或使用 Git commit hash
bash scripts/bump-version.sh $(git rev-parse --short HEAD)
```

统一更新所有 JavaScript 文件的版本号，避免缓存问题。

### 3. 部署前检查

```bash
bash scripts/pre-deploy.sh
```

运行完整的部署前检查：
- 语法检查
- 文件引用检查
- 版本号检查
- Git 状态检查
