# 🚀 启动本地测试服务器

## 快速启动（复制粘贴到终端）

```bash
cd /Users/bill/bazi-project
python3 -m http.server 8000
```

## 🌐 测试链接

启动服务器后，在浏览器中打开：

### 主页面（完整功能）
```
http://localhost:8000/index.html
```

### UI 拆分测试页面
```
http://localhost:8000/test-ui-split.html
```

## 📋 启动步骤

1. **打开终端**（Terminal）
2. **复制粘贴以下命令**：
   ```bash
   cd /Users/bill/bazi-project && python3 -m http.server 8000
   ```
3. **看到以下输出表示成功**：
   ```
   Serving HTTP on 0.0.0.0 port 8000 (http://0.0.0.0:8000/) ...
   ```
4. **保持终端窗口打开**，然后在浏览器访问上面的链接

## 🛑 停止服务器

在终端按 `Ctrl+C`

## ⚠️ 如果端口被占用

使用其他端口：
```bash
python3 -m http.server 3000
```

然后访问：`http://localhost:3000/index.html`
